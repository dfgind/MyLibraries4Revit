# MyLibraries4Revit

This package includes essential Revit API DLLs (RevitAPI.dll and RevitAPIUI.dll) bundled for reuse across .NET projects.

> ⚠️ You must have Revit installed to legally use these assemblies.

## Contents
- RevitAPI.dll
- RevitAPIUI.dll
